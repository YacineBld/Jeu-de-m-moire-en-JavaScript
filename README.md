# 🧠 Memory Game – Projet JavaScript interactif

Un **mini-jeu Memory (jeu de paires)** développé en **JavaScript Vanilla**, avec une **interface moderne**, un **timer en temps réel**, un **compteur de coups**, et une **animation de victoire**.

👉 Ce projet met en avant des compétences essentielles en **JavaScript front-end**, notamment la **manipulation du DOM**, la **gestion d’événements**, la **logique de jeu**, et la **création d’interfaces interactives**.

---

## 🚀 Aperçu du projet

Le joueur doit retrouver toutes les **paires de cartes identiques** en retournant les cartes deux par deux.

Le jeu inclut :

- un **plateau dynamique de 16 cartes** (8 paires),
- un **compteur de coups**,
- un **suivi du nombre de paires trouvées**,
- un **chronomètre en direct**,
- une **fenêtre de victoire** à la fin de la partie,
- un bouton pour **rejouer instantanément**.

Ce projet est idéal pour illustrer une **maîtrise propre du JavaScript natif** sans framework.

---

## ✨ Fonctionnalités

- 🃏 **Jeu de Memory interactif**
- 🔀 **Mélange aléatoire des cartes** à chaque partie
- 🖱️ **Retour des cartes au clic**
- ✅ **Détection automatique des paires**
- 🚫 **Blocage temporaire des interactions** pendant la vérification
- 📈 **Compteur de coups en temps réel**
- 🎯 **Suivi des paires trouvées**
- ⏱️ **Chronomètre intégré**
- 🎉 **Popup de victoire avec statistiques finales**

## 🧠 Compétences démontrées

Ce projet met en valeur plusieurs compétences importantes en développement front-end :

### Côté JavaScript
- Manipulation du DOM (`createElement`, `innerHTML`, `textContent`)
- Gestion des événements (`addEventListener`)
- Gestion d’état du jeu
- Logique conditionnelle
- Gestion des tableaux et duplication de données
- Mélange aléatoire de données (`shuffle`)
- Utilisation de `setTimeout` et `setInterval`
- Contrôle du flux utilisateur (verrouillage temporaire)
- Réinitialisation complète d’une interface dynamique

### Côté Front-end / UI
- Responsive design
- Animations CSS
- Effet de **flip 3D**
- Overlay de fin de partie
- Design moderne orienté portfolio
- Expérience utilisateur fluide et intuitive
